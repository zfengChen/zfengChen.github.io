面试高频题-css让超宽文本替换为省略号

发表于2022-02-06|更新于2022-02-07|面试高频题
|阅读量:

问题：

盒子设置了宽高，但是文本内容过长，文字会超出盒子显示。

目的：

让超出盒子部分的文字用省略号代替。

单行文本显示省略号

用到的三个属性分别是：overflow、text-overflow和white-space

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			.div1{
				width: 12.5rem;
				height:6.25rem;
				background-color: #ff007f;
				/* 隐藏超出范围的部分 */
				overflow: hidden;
				/* 文本溢出部分省略 */
				text-overflow: ellipsis;
				/* 所有文字在一行显示，不换行 */
				white-space: nowrap;
			}
		</style>
	</head>
	<body>
		<div class="div1">
			老师老师！！！最近面试我遇到多行的，虽然我搞懂了，但老师能不能
			搞一期 爱你爱你=替换省略号好像浏览器兼容性不太好是的，
因此有些公司会直接输入.....老师老师！！！
最近面试我遇到多行的，虽然我搞懂了，但老师能不能搞一期 爱你爱你
		</div>
	</body>
</html>

效果图：

多行文本显示省略号

方法一：

使用私有属性 -webkit-box

优点：简洁明了

缺点：私有属性（较大的局限性）

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			.div2 {
				width: 12.5rem;
				height: 6.8rem;
				background-color: #55ff7f;
				/* 隐藏超出范围的部分 */
				overflow: hidden;
				display: -webkit-box;
				/* 设置盒子内的子元素排列方向 */
				-webkit-box-orient: vertical;
				/* 设置从第几行开始省略 */
				-webkit-line-clamp: 5;
			}
		</style>
	</head>
	<body>
		<div class="div2">
			老师老师！！！最近面试我遇到多行的，虽然我搞懂了，
但老师能不能搞一期 爱你爱你替换省略号好像浏览器兼容性不太好是的，
因此有些公司会直接输入老师老师！！！最近面试我遇到多行的，虽然我搞懂了，
但老师能不能搞一期 爱你爱你
		</div>
	</body>
</html>

方法二：

双重伪元素

优点：符合css规范

缺点：背景如果是图片就没办法使用

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			.p3 {
				height: 210px;
				font-size: 31px;
				background-color: #55ff7f;
				/* 隐藏超出范围的部分 */
				overflow: hidden;
				position: relative;
				/* 文本对齐方式 */
				text-align: justify;
				padding-right: 1em;
			}
			.p3::before{
				content: '...';
				position: absolute;
				right: 0;
				bottom: 0;
			}
			.p3::after{
				content: '';
				display: inline;
				position: absolute;
				right: 0;
				width: 1em;
				height: 1em;
				background-color: red;
				margin-top: 0.5em;
			}
		</style>
	</head>
	<body>
		<p class="p3">
			老师老师！！！最近面试我遇到多行的，虽然我搞懂了，
但老师能不能搞一期 爱你爱你替换省略号好像浏览器兼容性不太好是的，
因此有些公司会直接输入老师老师！！！最近面试我遇到多行的，
虽然我搞懂了，但老师能不能搞一期 爱你爱你老师老师！！！
		</p>
	</body>
</html>

当盒子宽度够宽，能显示全部文字时，不需要省略号了，和背景色一样颜色的正方形小方块就会遮住省略号。

方法三：（一般面试不问）

linear-gradient，不使用省略号

优点：符合css规范

缺点：还是背景问题

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			.p4 {
				height: 210px;
				overflow: hidden;
				text-align: justify;
				position: relative;
				background-color: #55ff7f;
				font-size: 31px;
			}
			.p4::after{
				content: '';
				position: absolute;
				right: 0;
				bottom: 0;
				height: 1.2em;
				width: 20%;
				/* 使用渐变背景色来代替省略号，体现美观 */
				background: linear-gradient(to right,rgba(255,255,255,0),#ff55ff 80%);
				margin-bottom: 0.2em;
			}
		</style>
	</head>
	<body>
		<p class="p4">
			老师老师！！！最近面试我遇到多行的，虽然我搞懂了，
但老师能不能搞一期 爱你爱你替换省略号好像浏览器兼容性不太好是的，
因此有些公司会直接输入老师老师！！！最近面试我遇到多行的，虽然我搞懂了，
但老师能不能搞一期 爱你爱你老师老师！！！
		</p>
	</body>
</html>

这样一来就会体现一种朦胧美感，这渐变色要设置成和背景同色。
