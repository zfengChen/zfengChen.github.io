前言

项目已上线，现在可以通过访问基于vue的校园实时防疫监控来使用。

基于vue的校园实时防疫监控

知识点

  1. Vue基础知识
  2. Axios网络请求
  3. Echarts可视化
  4. Swiper焦点轮播图
  5. Vue-Router
  6. 封装组件

可使用地址

用友：https://api.yonyoucloud.com/apilink/#/
天行：https://www.tianapi.com/

项目运行

  1. 安装依赖
    npm i
  2. 开发环境下运行
    npm run serve
  3. 或者生产环境下运行
    // 安装依赖
npm i -g serve
// 项目打包
npm i build
// 项目本地预览
serve -s dist

效果图






开发过程中的思维方式

  1. 在Vue框架中，是组件式开发，将组件分离的更加细致一点

关于组件实例化问题

  1. 拆分中组件之后，组件的多次调用，会每一次实例化一个独立的对象

未解决问题

  1. 跨域问题(主要由后端解决)
    • 开发环境
      开发环境下的跨域处理，只能在开发环境下运行，打包之后是不能运行的
      proxy都是开发环境跨域
    • 生产环境
      cors(主)
      jsonp(次)
  2. 打包路径问题
    publicPath指定具体子路径信息 
    publicPath: process.env.NODE_ENV === 'production' ? '/covid19' : '/'
