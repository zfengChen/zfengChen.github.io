npm安装package.json中的模块依赖_致敬code大神的博客-CSDN博客

npm一键安装package.json里的所有依赖文件

npm install

dependencies（核心依赖/运行依赖 基本上是第三方包）

只安装核心依赖项的指令

npm install --dependencies

devDependencies（开发依赖）

npm install --devDependencies
