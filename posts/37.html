面试高频题-css外边距塌陷

发表于2022-02-07|更新于2022-02-07|面试高频题
|阅读量:

前言

什么是外边距塌陷？

假设A元素和B元素想上下隔开40px的距离，设置A元素的下外边距（margin-bottom：20px），B元素的上外边距（margin-top：20px），但是实际的间隔却只有20px。这就是外边距塌陷。

为什么要故意设置外边距塌陷？

当存在多段文本设置了相同的上下外边距，为了使每段文本的间隙都相等，整体很整齐，这就是外边距塌陷的初衷。

总结：为了解决段落之间垂直方向上的空隙。

什么时候会发生外边距塌陷？

1垂直方向,不是水平方向;

2,块级元素,不是行内元素,也不是行内块级元素。

外边距计算

1、正数 && 正数 最大的数

2、负数 && 负数 绝对值最大的数

3、正数 && 负数 相加的和

常见的外边距塌陷情况

相邻元素（兄弟元素）

相邻元素外边距塌陷情况，给A元素设置50px的下外边距，给B元素设置100px上外边距，我们以为中间间距为150px，实际上只有100px，产生这种情况的原因可以参考外边距计算

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }
        .box1 {
            height: 100px;
						text-align: center;
						line-height: 100px;
            background-color: #ccc;
            margin-bottom: 50px;
        }

        .box2 {
            height: 100px;
						text-align: center;
						line-height: 100px;
            background-color: #FAE8C8;
            margin-top: 100px;
        }
    </style>
</head>

<body>
    <div class="box1">
		<h1>A元素</h1>
	</div>
	<div id="fa">
		<div class="box2">
			<h1>B元素</h1>
		</div>
	</div>
</body>

</html>

解决方法：

给任意一个元素添加一个父元素，然后设置overflow为hidden。

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }

        .box1 {
            height: 100px;
						text-align: center;
						line-height: 100px;
            background-color: #ccc;
            margin-bottom: 50px;
        }

        .box2 {
            height: 100px;
						text-align: center;
						line-height: 100px;
            background-color: #FAE8C8;
            margin-top: 100px;
        }
				#fa{
					overflow: hidden;
				}
    </style>
</head>

<body>
    <div class="box1">
		<h1>A元素</h1>
		</div>
	<div id="fa">
		<div class="box2">
			<h1>B元素</h1>
		</div>
	</div>
</body>

</html>

父子元素（嵌套元素）

父子元素外边距塌陷情况，我们想子元素距离父元素的顶部100px，所以给子元素设置上边距100px。但实际情况确实父元素距离浏览器顶部100px，子元素还在父元素的顶部。产生这种情况的原因也和外边距计算有关。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        *{
            margin: 0;
            padding: 0;
        }
        .father{
            width: 200px;
            height: 200px;
            background-color: red;
        }
        .son{
            width: 100px;
            height: 100px;
            background-color: #ccc;
            margin-top: 100px;
        }
    </style>
</head>
<body>
    <div class="father">
        <div class="son"></div> 
    </div>
  
</body>
</html>

解决方法有三个，分别是：

  • 给父元素加border

    .father{
      width: 200px;
      height: 200px;
      background-color: red;
			border: 1px solid black;
}

  • 给父元素加padding

    .father{
      width: 200px;
      height: 200px;
      background-color: red;
			padding: 1px;
}

  • 给父元素设置overflow:hidden;成独立渲染区域

    .father{
      width: 200px;
      height: 200px;
			background-color: red;
			overflow: hidden;
}

总结：解决外边距塌陷的常见方法

绝对定位

当遇到以下这种情况，A元素（box1）是红色，B元素（box3）是灰色，C元素（box2）是蓝色，B和C是父子关系，A和C是兄弟关系。我们想B元素距离C元素的顶部有20px的高度。于是直接给B元素加上 margin-top:20px，这时候就会出现外边距塌陷。

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			*{
				margin: 0;
				padding: 0;
			}
			.box1{
				height: 100px;
				background-color: red;
			}
			.box2{
				height: 100px;
				background-color: blue;
			}
			.box3{
				height: 20px;
				width: 100%;
				background-color: #ccc;
				margin-top: 20px;
			}
		</style>
	</head>
	<body>
		<div class="box1">A元素</div>
		<div class="box2">
			<div class="box3">B元素</div>
		</div>
	</body>
</html>

解决方法：使用绝对定位，给C元素（box2）添加相对定位relative，给B元素（box3）添加绝对定位absolute。

.box2{
				height: 100px;
				background-color: blue;
				position: relative;
			}
.box3{
				position: absolute;
				height: 20px;
				width: 100%;
				background-color: #ccc;
				margin-top: 20px;
			}

行内块级元素

首先行内块元素不会发生外边距塌陷，我们还是用之前的例子，我们可以给B元素（box3）添加 display:inline-block 

.box3{
				display: inline-block;
				height: 20px;
				width: 100%;
				background-color: #ccc;
				margin-top: 20px;
			}

注意：该方法需要给行内块元素设置宽高

相对定位

该方法给B元素（box3）设置相对定位，通过边偏移（top: 20px）来代替外边距（margin-top: 20px）

.box3{
				position: absolute;
				top: 20px;
				height: 20px;
				width: 100%;
				background-color: #ccc;
			}

注意：该方法使用了相对定位（position: absolute;）后，不能再使用上外边距（margin-top: 20px;）了，如果再使用上外边距（margin-top: 20px;），还是会出现外边距塌陷。因为相对定位是相对原来的位置进行偏移，原来区域还是存在的。

浮动

通过给B元素添加向左浮动或者向右浮动，再设置上外边距（margin-top: 20px;），就可以解决外边距塌陷问题。

.box3{
				float: left;
				height: 20px;
				width: 100%;
				background-color: #ccc;
				margin-top: 20px;
			}

注意：该方法需要考虑元素在其父元素的位置，是在左侧还是右侧还是中间，如果元素在父元素的中间就比较麻烦，毕竟没有向中间浮动的设置。

BFC

创建BFC的元素不会和他的子元素发生外边距塌陷的情况

创建BFC的四种方法是：overflow、float、position、display。

这里我们给C元素添加overflow:hidden，来解决外边距塌陷

.box2{
				height: 100px;
				background-color: blue;
				overflow: hidden;
			}
.box3{
				height: 20px;
				width: 100%;
				background-color: #ccc;
				margin-top: 20px;
			}

注意：overflow:hidden有时候会隐藏掉溢出的部分，有局限性。

内边距

既然不能设置外边距（margin-top: 20px），于是我们可以设置内边距（padding-top: 20px）

.box3{
				height: 20px;
				width: 100%;
				background-color: #ccc;
				padding-top: 20px;
			}

注意：记得要把之前的设置外边距（margin-top: 20px）去掉，不然会出现内外边距叠加现象。

边框

给C元素加border，只要有东西把B元素压回C元素内，就不会出现外边距塌陷情况。

.box2{
				height: 100px;
				background-color: blue;
				border-top: 5px solid magenta;
			}
.box3{
				height: 20px;
				width: 100%;
				background-color: #ccc;
				margin-top: 20px;
			}

空元素

特殊情况，当我们想给两个元素添加一点空隙的时候我们可以在两元素之间创建一个空元素，给这个空元素添加一个上外边距。

.divEmpty{
				margin-top: 20px;
}
