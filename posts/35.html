面试高频题-css水平居中

发表于2022-02-06|更新于2022-02-06|面试高频题
|阅读量:

传统网页布局的三种定位方式：

  • 普通流
  • 浮动
  • 定位

普通流（inline）行内元素 居中

普通流里常用 text-align：center 居中显示元素。注意：这种情况下，块级元素可以实现居中。但是行内元素不行。

解决方法：

行内元素通过给父级元素添加 text-align：center 使行内元素居中显示。

这是直接给行内元素添加 text-align：center 的情况：

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			*{
				color: red;
			}
			p{
				text-align: center;
			}
			h1{
				text-align: center;
			}
			span{
				text-align: center;
			}
		</style>
	</head>
	<body>
		<p>this is p tag</p>
		<h1>this is h1 tag</h1>
		<span>this is span tag</span>
	</body>
</html>

这是给行内元素的父元素添加了 text-align: center

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			*{
				color: red;
			}
			p{
				text-align: center;
			}
			h1{
				text-align: center;
			}
			#span-fa{
				text-align: center;
			}
		</style>
	</head>
	<body>
		<p>this is p tag</p>
		<h1>this is h1 tag</h1>
		<div id="span-fa">
			<span>this is span tag</span>
		</div>
	</body>
</html>

普通流（block）块级元素 居中

块级元素可以设置宽高，所以我们可以调整外边距 margin 来实现元素居中显示。

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			.div1{
				width: 200px;
				height: 200px;
				background-color: red;
				margin: 0 auto;
			}
		</style>
	</head>
	<body>
		<div class="div1"></div>
	</body>
</html>

注意：ie5 和 ie6 不支持这种居中方法。

解决方法：给 body 添加 text-align: center; 给不需要居中的元素添加text-align: left;（元素默认左对齐）

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			body{
				text-align: center;
			}
			h1{
				text-align: left;
			}
			.div1{
				width: 200px;
				height: 200px;
				background-color: red;
				margin: 0 auto;
			}
		</style>
	</head>
	<body>
		<h1>this is h1 tag</h1>
		<div class="div1"></div>
	</body>
</html>

浮动元素 居中

浮动元素脱离了普通文档流 margin: 0 auto; 已经失效。所以浮动元素可以通过添加相对定位再配合外边距来实现元素居中显示。具体操作如下：

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">	
			.div2{
				width: 200px;
				height: 200px;
				background-color: blue;
				float: left;
				position: relative;
				left: 50%;
				margin-left: -100px;
			}
		</style>
	</head>
	<body>
		<div class="div2"></div>
	</body>
</html>

注意：该方法需要元素有一定的宽度，以及知道该元素的宽度。

可以使用css3优化该方法：（使用transform来移动x轴的位置）

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">	
			.div2{
				width: 200px;
				height: 200px;
				background-color: blue;
				float: left;
				position: relative;
				left: 50%;
				transform: translateX(-50%);
			}
		</style>
	</head>
	<body>
		<div class="div2"></div>
	</body>
</html>

注意：这个经过css3优化的方法，需要考虑兼容性的问题

绝对定位 居中

方法一：

如果是一个行内元素，设置了绝对定位，那么这个行内元素就可以设置宽度和高度。所以我们可以使用普通流块级元素居中的方法： margin： 0 auto；

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			body{
				text-align: center;
			}
			.span2{
				position: absolute;
				right: 0;
				left: 0;
				margin: 0 auto;
			}
		</style>
	</head>
	<body>
		<span class="span2">1111111</span>
	</body>
</html>

方法二：

已知元素的宽度和高度，可以通过设置 左边偏移 left:50%**； 配合设置 **左外边距 margin-left 为宽度的一半，注意该数值必须为负数。

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			body{
				text-align: center;
			}
			.span2{
				width: 120px;
				position: absolute;
				right: 0;
				left: 0;
				left: 50%;
				margin-left: -60px;
			}
		</style>
	</head>
	<body>
		<span class="span3">222</span>
	</body>
</html>

方法三：

通过css3来优化方法二，使用 transform 属性。

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			body{
				text-align: center;
			}
			.span2{
				width: 120px;
				position: absolute;
				right: 0;
				left: 0;
				left: 50%;
				transform: translateX(-50%);
			}
		</style>
	</head>
	<body>
		<span class="span3">222</span>
	</body>
</html>

注意：css3的方法，需要考虑兼容性的问题

总结：以上是最基础的居中方法，还有其他的居中方法，比如添加父元素等。
