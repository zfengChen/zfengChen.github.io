发送Ajax请求的四种常用方式

发表于2022-02-01|更新于2022-02-01|笔记
|阅读量:

前言

Ajax简介

  • AJAX = Asynchronous JavaScript and XML（异步的 JavaScript 和 XML）。

  • AJAX 不是新的编程语言，而是一种使用现有标准的新方法。

  • AJAX 最大的优点是在不重新加载整个页面的情况下，可以与服务器交换数据并更新部分网页内容。

  • AJAX 不需要任何浏览器插件，但需要用户允许JavaScript在浏览器上执行。

Ajax 应用

  • 运用 XHTML+CSS 来表达资讯;

  • 运用 JavaScript 操作 DOM 来执行动态效果;

  • 运用 XML 和 XSLT 操作资料;

  • 运用 XMLHttpRequest 或新的 Fetch API 与网页服务器进行异步资料交换;

发送Ajax请求的四种常用方式

1-原生Ajax

发送Ajax请求

在需要发送请求的页面书写

注意要清楚知道想要发送请求的对象，比如：点击按钮发送Ajax请求，那么就是按钮点击事件触发的发送Ajax请求。

get请求

// 1、创建对象
      var xhr = new XMLHttpRequest();
      // 2、初始化 设置请求方法和 url
      xhr.open('get', 'http://127.0.0.1:3000/server?a=100&b=200&c=300');
      // 3、发送
      xhr.send();
      //4. 事件绑定 处理服务端返回的结果
      // on  when 当....时候
      // readystate 是 xhr 对象中的属性, 表示状态 0 1 2 3 4
      // change  改变
      xhr.onreadystatechange = () => {
          //判断 (服务端返回了所有的结果)
          if (xhr.readyState === 4) {
              //判断响应状态码 200  404  403 401 500
              // 2xx 成功
              if (xhr.status >= 200 && xhr.status < 300) {
                  //处理结果  行 头 空行 体
                  //响应 
                  // console.log(xhr.status);//状态码
                  // console.log(xhr.statusText);//状态字符串
                  // console.log(xhr.getAllResponseHeaders());//所有响应头
                  // console.log(xhr.response);//响应体
                  //设置 result 的文本
                  res.innerHTML = xhr.response;
              }
          }
      }

post请求

//1. 创建对象
      const xhr = new XMLHttpRequest();
      //2. 初始化 设置类型与 URL
      xhr.open('post', 'http://127.0.0.1:3000/server');
      //设置请求头
      // Content-Type 设置请求体内容的类型
      // application/x-www-form-urlencoded 
      // 设置 请求体向服务器传递参数的类型
      xhr.setRequestHeader('Content-Type', 'application/x-www-form-urlencoded');
      // 自定义请求头信息 上面的是预定义的请求头
      xhr.setRequestHeader('name', 'atguigu');

      //3. 发送
      // xhr.send();
      // 设置请求体 向服务器传递参数
      xhr.send('a=100&b=200');

      //4. 事件绑定
      xhr.onreadystatechange = () => {
          //判断
          if (xhr.readyState === 4) {
              if (xhr.status >= 200 && xhr.status < 300) {
                  //处理服务端返回的结果
                  res.innerHTML = xhr.response;
              }
          }
      }

get请求和post请求区别

get请求的请求参数可以直接拼接在url地址上

post请求的请求参数是放在send（）方法里的

  • post请求如果要发送自定义的请求头信息 要自己通过setRequestHeader（）方法设置

    // 自定义请求头信息 上面的是预定义的请求头

xhr.setRequestHeader('name', 'atguigu');

  • post请求还要设置请求体向服务器传递参数的类型

    xhr.setRequestHeader('Content-Type', 'application/x-www-form-urlencoded');

2-jQuery发送Ajax请求

jq发送Ajax请求有两种方法

第一种

// 此处以get请求为例 get('请求地址'，'请求参数'，'回调函数'，'响应体的类型')
// get请求
$.get('http://127.0.0.1:3000/jq', { a: 100, b: 200 }, (res) => {
                console.log(res);
            }, 'json');
// post请求
$.post('http://127.0.0.1:3000/jq', { a: 100, b: 200 }, (res) => {
                console.log(res);
            }, 'json');

注意设置了响应体的类型后，后台服务器代码要设置发送json格式的字符串数据

// 后台服务器代码
// 1、引入express
var express = require('express');
// 路径处理模块
var path = require('path');
//  2、创建应用对象
var app = express();

// 静态资源访问服务功能
app.use(express.static(path.join(__dirname, 'public')));

// 3、jq服务 http://127.0.0.1:3000/jq
app.all('/jq', (req, res) => {
    //设置响应头  设置允许跨域
    res.setHeader('Access-Control-Allow-Origin', '*');
    // 设置 自定义请求头
    res.setHeader('Access-Control-Allow-Headers', '*');
		// 设置返回的数据
    const data = {
        name: 'admin'
    };

    //设置响应体 返回json格式的字符串数据
    res.send(JSON.stringify(data));

})

//4. 监听端口启动服务
app.listen(3000, () => {
    console.log('running...');
})

第二种

// 通用方式发送Ajax请求
$.ajax({
                // url 地址
                url: 'http://127.0.0.1:3000/jq',
                // 发送的 参数
                data: { a: 100, b: 200 },
                // 请求类型
                type: 'get',
								// 设置发送 参数 的数据类型 默认值
								// 如发送json数据 改成 application/json 
								contentType: 'application/x-www-form-urlencoded',
                // 设置 响应体结果 类型
                dataType: 'json',
                // 成功的回调
                success: (data) => {
                    console.log(data);
                },
                // 超时时间
                timeout: 2000,
                // 失败回调
                error: () => {
                    console.log('出错了');
                },
                // 头信息
                headers: {
                    c: 300,
                    d: 400
                }
            });

3-Axios发送Ajax请求

axios中文文档|axios中文网

Axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中。是一个热门的Ajax工具库。

发送get、post请求（单一）

// 对url地址进行简化
axios.defaults.baseURL = 'http://127.0.0.1:3000';

// 发送get请求
axios.get('/axios', {
                // 设置url参数
                params: {
                    id: 100,
                    vip: 7
                },
                //请求头信息
                headers: {
                    name: 'admin',
                    age: 20
                }
            }).then((value) => {
								// 返回的结果
                console.log(value);
            });

// 发送post请求
axios.post('/axios', {
                username: 'admin',
                password: '123456'
            }, {
                //url 
                params: {
                    id: 200,
                    vip: 9
                },
                //请求头参数
                headers: {
                    height: 180,
                    weight: 180,
                }
            });

通用方法（这样子不同请求可以不用分开写）

axios({
                //请求方法
                method: 'POST',
                //url
                url: '/axios',
                //url参数
                params: {
                    vip: 10,
                    level: 30
                },
                //头信息
                headers: {
                    a: 100,
                    b: 200
                },
                //请求体参数
                data: {
                    username: 'admin',
                    password: 'admin'
                }
            }).then(response => {
                //响应状态码
                console.log(response.status);
                //响应状态字符串
                console.log(response.statusText);
                //响应头信息
                console.log(response.headers);
                //响应体
                console.log(response.data);
            })

4-fetch函数发送Ajax请求

Fetch API - Web API 接口参考 | MDN

fetch简介

  • fetch是web提供的一个可以获取异步资源的api，目前还没有被所有浏览器支持。
  • fetch提供的api返回的是Promise对象，如果不知道Promise和它的用法的朋友们可以看我下一篇文章，有关Promise的相关介绍以及用法。

fetch的特点

  • fetch() 必须接受一个参数——资源的路径。无论请求成功与否，它都返回一个 Promise 对象，resolve 对应请求的 Response。
  • fetch() 是全局对象的
  • fetch() 不会接受跨域 cookies；你也不能使用 fetch() 建立起跨域会话，其他域的Set-Cookie头部字段将会被无视。
  • fetch 不会发送 cookies。除非你使用了credentials 的初始化选项。（自 2017 年 8 月 25 日以后，默认的 credentials 政策变更为 same-origin。Firefox 也在 61.0b13 版本中进行了修改）
// 使用fetch函数发送ajax请求
fetch('http://127.0.0.1:3000/fetch', {
    //请求方法
    method: 'POST',
    //请求头
    headers: {
        name: 'admin'
    },
    //请求体
    body: 'username=admin&password=123456'
}).then(response => {
    // 用 then的回调 处理回调结果
    // 获取 响应体 结果 用text（）方法
    // 若服务端返回的结果是json格式的数据 则用json（）方法
    return response.text();
    // return response.json();
    }).then(response => {
      console.log(response);
    });
